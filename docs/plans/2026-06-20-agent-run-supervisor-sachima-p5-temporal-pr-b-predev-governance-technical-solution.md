## Control-flow design

Happy-path pipeline (resolves open questions #2, #3):

1. **Admission** ‚Äî `execute(...)` checks flag + exact token. Off/mismatch ‚Üí `runtime_disabled`/`runtime_approval_mismatch`/`runtime_precondition_unmet`; **no Temporal call**; caller stays on local/offline baseline.
2. **Translate** ‚Äî executor maps WP4 `request`/`role_binding`/`resolved_inputs` ‚Üí sanitized `StartRequest` via `contracts.py`. **Translation lives in `step_executor.py`; `workflow.py` never imports `ai_flow_executor`/WP4 internals** (OQ#2).
3. **Dispatch** ‚Äî `control_surface.start(...)` (no-throw) ‚Üí `runtime_client` uses the **caller-supplied connected client** to start `StepWorkflow` under a deterministic workflow id from sanitized refs + `idempotency_material` (OQ#3 ‚Äî client never self-connects, owns no lifecycle).
4. **Run** ‚Äî `@workflow.run` drives a controlled-deterministic `step_activity`; artifacts surface as **claim-check refs only**.
5. **Observe** ‚Äî `query` ‚Üí sanitized `QuerySnapshot`; `update` accepts only `{resume, request_cancel}`, event-key idempotent.
6. **Terminalize** ‚Äî `close`/completion ‚Üí sanitized terminal projection.
7. **Return** ‚Äî `StepExecutionOutcome` with claim-check refs; **no business verdict inferred from success.**

Alternate flows: duplicate-identical ‚Üí replayed projection, nothing launched twice; duplicate-divergent ‚Üí fail closed (`runtime_idempotency_conflict`/`invalid_start_payload`), reconciliation compares sanitized canonical payloads ‚Äî **public digests are evidence, not a trust boundary** (OQ#4); restart/recovery ‚Üí `recover` reattaches **by workflow id**, no-throw, **never auto-relaunches uncertain work**; cancel ‚Üí cooperative, preserves WATCH (below).

## No-leak and determinism strategy

**No-leak (FR7)** ‚Äî two merge-blocking scans + canary:
- **SCAN 1 (JSON projection):** walk durable records / query snapshots / evidence JSON; assert every key ‚àà allowlist and no value matches forbidden patterns.
- **SCAN 2 (serialized event-history bytes):** fetch **real** hermetic history via `runtime_client.serialized_history_bytes`; scan **both** the `json.loads`-decoded structure **and** the raw bytes against forbidden byte patterns.
- **Canary sentinels:** seed unique sentinels into the controlled body's raw prompt/context inputs; assert they appear in **neither** projection **nor** serialized history ‚Äî positive proof the claim-check boundary holds. Any hit ‚Üí blocker ‚Üí `runtime_history_leak_detected`.

**Determinism (FR3/FR8):** workflow code is pure (no I/O, wall-clock, randomness, client calls); non-determinism lives in activities/outside. A **determinism replay gate** re-runs recorded histories through `temporalio.worker.Replayer`; any non-determinism error blocks merge.
